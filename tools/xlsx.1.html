NAME

xlsx - concatenate and print xlsx file

SYNOPSIS

xlsx [option] xlsx-file ...

DESCRIPTION

xlsx print content of SpreadsheetML of Office Open XML, also known Microsoft Excel (.xlsx) format.
Without arguments, xlsx shows quick help.

FUNCTIONS

  • -f, --format "format"
    Set output format as "csv" or "tsv". (default: "tsv")

  • -i, --index index
    Set index of sheets (zero based). (default: 0)

  • --sep "sep"
    Set a separator of piped input stream. (default: " ")

  • -h, --help
    Show help.

  • -v, --version Print the version.

NOTES

  • Date string formatting is very lazy. You should set custom date format like "yyyy-mm-dd". Acceptable format is below. Day of the week is represented as English.

    • yyyy (years like 2006)
    • yy (years like 06)
    • mmmm (months like January)
    • mmm (months like Jan)
    • mm (months like 01)
    • m (months like 1)
    • dd (date like 02)
    • d (date like 2)
    • aaaa (days like Monday)
    • aaa (days like Mon)
    • hh (hours like 15)
    • h (hours like 3)
    • :mm (minutes like 04)
    • :ss (seconds like 05)
